### Routing & guards
- Single login at **/login**. After login → redirect by role:
  - admin → **/admin/dashboard**
  - agent → **/agent/dashboard**
- **/signup**: only for **agent** registration. Admin accounts are created from Admin > Users.
- **/forgot-password**: mock email flow.
- **/admin/** routes require role==='admin'. If not, show **403 (Forbidden)** with a button “Go to my panel”.
- **/agent/** routes require role==='agent'. Wrong role → 403.
- **/verify?code=FAS-XXXXXX**: certificate verification page.
- 404 page for unknown routes.

### Types (src/types.ts)
Define:
- Role = 'admin' | 'agent'
- User { id, name, email, role, agencyId? }
- Agency { id, name, logoUrl?, address?, lat?, lng?, staffSize?, annualStudents?, website?, phone?, primaryContactName?, primaryContactEmail? }
- Course { id, title, slug, sections: Section[] }
- Section { id, title, lessons: Lesson[] }
- Lesson { id, title, html, quizId? }
- Quiz { id, title, questions: Question[], passPercent, isFinal? }
- Question = MCQ | BooleanQ
- MCQ { id, type:'mcq', text, options:string[], answerIndex:number }
- BooleanQ { id, type:'boolean', text, answer:boolean }
- Progress { userId, courseId, percent, lessonCompletedIds:string[] }
- Attempt { id, userId, quizId, scorePercent, correct, incorrect, date }
- Certificate { id, userId, courseId, scorePercent, code, issuedAt }
- PaymentProvider = 'none'|'stripe'|'iyzico'|'paytr'|'paypal'
- PaymentStatus = 'unpaid'|'pending'|'paid'|'failed'|'refunded'
- Order { id, userId, courseId?, title, currency:'USD'|'EUR'|'TRY', amount:number, status:PaymentStatus, provider:PaymentProvider, paymentIntentId?, createdAt, updatedAt }
- IntegrationConfig {
    id; name:'Generic Webhook'|'n8n'|'Kommo'|'Google Sheets'|'SMTP Mail'|'Custom API';
    enabled:boolean;
    endpointUrl?, apiKey?, webhookSecret?;
    smtpHost?, smtpPort?, smtpUser?, smtpPass?, fromEmail?;
    sheetId?, tabName?;
    kommoDomain?, kommoToken?;
    n8nWorkflowId?;
    lastTestAt?, lastTestStatus?:'ok'|'fail';
  }
- PaymentConfig { enabled:boolean; provider:PaymentProvider; publicKey?; secretKey?; successUrl?; cancelUrl?; webhookSecret? }
